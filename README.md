# ☕ CoffeeShopBot (Telegram Bot + Web Admin Panel)

Интерактивная система начисление бонусов для кофейни со встроенной административной панелью.

## Технологический стек:
- **Бэкенд:** C# (ASP.NET Core)
- **База данных:** SQLite (с использованием Entity Framework Core)
- **Фронтенд админки:** HTML5, CSS3 (Bootstrap), JavaScript Fetch API
- **API мессенджера:** Telegram.Bot Library

## Функционал:
- Автоматическая регистрация пользователя по номеру телефона при старте.
- Информация о кофейне через кнопки меню бота (О кофейне, баланс, контакты).
- Автоматическое сообщение от бота при списании/начислении бонусов.
- Административная веб-панель для начисления и списания бонусов.
- Безопасность: Админка защищена Cookie-авторизацией.
